What is a programming language?

Programming is a way for us to give instructions to computers. We give it an instruction manual and the computer follows. Without programming codes, computers cannot perform any task. Just like humans, computers need a language of communication, being based on zeros and ones. To understand the communication of computers from zeros and ones for us will be very difficult or even impossible. Programming language is between computer language and human language to create communication between them. The codes are created in files but are not understandable by the computer. For this we need a translator who understands the codes created by humans and the communication language of the computers. This translator is another human-made program called an interpreter. Interpreter is a program that can analyse and execute a program line by line. For this tutorial I chose the Python interpreter. Go to www.python.org and follow the instructions to download the python interpreter to be able to create and execute codes written in this programming language. To create Python files with codes we need IDE (Integrated Developer Environment) like PyCharm, VS, and Atom which provides a wide range of essential tools for Python developers.


Python data types.

A data type is a value in Python

  • int - It contains positive or negative whole numbers ex:...,-1, -2, 0, 1, 2,...
  • float - Decimal numbers ex: 1.4, 17.23, 125.12...
  • bool - It's used to represent the truth value of an expresion ex:(True or False)
  • str - Sequence of character data ex:("hello")
  • list - Is a collection of data that can be change ex:[2, 'a', 5.7]
  • tuple - Are used to store collections of data cannot be change ex: (3, 6.7, 'w')
  • set - A collection of data which is unchangeable ex: {2, 4, 7}
  • dict - Has value and key ex: {1:'a', 2:'b', 4:'z'}

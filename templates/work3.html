What is Cloud Computing?

Cloud computing has been used in IT industry for many years. Cloud computing is a remote, virtual pool of on-demand, shared resourses that can be rapidly deployed at scale. Cloud computing is based on virtualization. Virtualization allows you to create multiple virtual machines or VMs, where all of them run on single server. Each of VM has it's own operating system and set of applications. VMs are created by Hypervisor. Hypervisor is a software used to create the virtualized environment which is located between physical hardware and the virtual machines, to create a share pool of virtual hardware resources. Virtual machines are also called instance or server.

The benefits of virtualization are:
  • Easy share resources - Virtualization makes it much easier for many different people to share the same hardware. On VM you can run Windows or Linux operating systems.
  • Less waste - Most computers are not constantly running 100% utilization. Right now your PC probably using a small fraction of it's CPU and memory to run this web page. Virtualization gives you access to previously unused resources.
  • Lower costs - Virtualization enables better use of physical resources, which increases agility and enables IT flexibility and scalability. In this way it is possible to reduce operating expenses, as well as maintenance of physical hardware.
  • More environmentaly friendly - Less hardware that need to be manufactured and less power consumed which means that virtualization helps promote green computing.

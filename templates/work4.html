What is Web Development?


Web development is the building and maintenance of websites; it is a whole process of creating web pages and maintaining them. Web pages are created by web developers with the help of programming languages ​​and software.

HTML Fundamentals


HTML(Hyper Text Markup Language) is a markup language for creating Web Pages. HTML describes the structure of a Web page which consists of a series of elements. Those elements tell the browser how to display the content.
In order to access an HTML file across the internet, its web address must be entered into the address field of the web browser. The web address is known as its "Uniform Resourse Locator" or URL which has three parts:
Protocol - any URL using the HTTP protocol begins by specifying the protocolas http:// or secure htpps://
Domain - the host name of the computer from which the file can be downloaded. For exemple: www.examples.com
Path - the file name prefixed by any parent directory names where applicable. For instance: /htdocs/index.html.
